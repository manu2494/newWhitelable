Title

The title component shows the title of the page, or a custom title based on standard HTML header tags (h1,h2,h3,h4,h5,h6)

It has differing sizes for desktop, tablet and mobile

Available Styles

The following groups are available to style the header in the template

  • Alignment
    • Left (align-left)
    • Center (align-center) 
    • Right (align-right)
  • Case
    • Uppercase (case-upper)
    • Lowercase (case-lower)
    • Captitalize (case-capitalize)
  • Link Style
    • None link-nostyle)
  • Color
    • Primary (color-primary)
    • Secondary (color-secondary)
    • Tertiary (color-tertiary)
  • Background
    • Primary (background-theme-1)
    • Secondary (background-theme-2)
    • Tertiary (background-theme-3)
  • Font
    • Primary (font-primary)
    • Secondary (font-secondary)
    • Tertiary (font-tertiary)
  • Weight
    • Normal (weight-normal)
    • Medium (weight-medium)
    • Heavy (weight-heavy)
  • Line Height
    • Small (height-small)
    • Medium (height-medium)
    • Large (height-large)

Dialog Options

The following items are available in the Title dialog.

  • Title
    • Defaults to the page title if left empty
  • ID
    • A unique HTML ID for the title
